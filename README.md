# 🖌️ Baddies in Terminal

![screenshot](./README_ASSETS/banner.png)

A simple repository for compiling terminal color schemes (Base16), ready to be applied.

🖥️ Supported terminals:
- 🐱 Kitty
- 🚀 Alacritty
- 🐜 Termite
- X  XResources
- 📱 Termux
- 🪟 Windows Terminal
- 🐧 Linux TTY
